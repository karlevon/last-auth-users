# 📊 Last Auth Users — Яндекс 360

Скрипт для сбора данных о последних авторизациях пользователей организации Яндекс 360.

## ✨ Возможности

- 📡 Получение событий авторизации через Audit Log API Яндекс 360
- 👤 Определение последней успешной авторизации каждого пользователя
- 💻 Определение платформы входа (браузер, мобильное приложение, OAuth-токен и др.)
- 🌐 Сохранение IP адреса последней авторизации
- 📥 Выгрузка результатов в CSV файл
- 📄 Поддержка пагинации для больших организаций

## 📋 Поддерживаемые типы авторизаций

| Тип события | Платформа |
|---|---|
| `id_account.login` | Браузер |
| `id_account.auth` | Браузер |
| `id_cookie.set` | Браузер |
| `id_device_token.issued` | Мобильное приложение |
| `id_nondevice_token.issued` | OAuth-токен |
| `id_app_password.login` | Пароль приложения |

## ⚙️ Настройка

Открой файл `last_auth_users.py` и заполни параметры в начале файла:

```python
ORG_ID      = ""              # ID организации Яндекс 360
TOKEN       = ""              # OAuth токен
DATE_FROM   = "2026-05-18"    # Начало периода (YYYY-MM-DD)
DATE_TO     = "2026-05-22"    # Конец периода  (YYYY-MM-DD)
OUTPUT_FILE = "last_logins.csv"  # Имя выходного файла
PAGE_SIZE   = 100             # Размер страницы (макс. 100)
